# Mini-Bioinformatics Pipeline & Reporting

A reproducible Snakemake pipeline for Quality Control (QC) of long-read (Oxford Nanopore) sequencing data. The pipeline runs NanoPlot for comprehensive QC, computes per-read statistics (GC content, read length, mean quality), and generates distribution plots with summary statistics.

---

## Repository Structure

```
├── Snakefile                 # Snakemake pipeline definition
├── environment.yml           # Conda environment specification
├── data/
│   └── barcode77.fastq       # Raw ONT long-read FASTQ input
├── scripts/
│   ├── read_stats.py         # Per-read statistics calculator
│   └── visualize.py          # Distribution plots & summary stats
└── results/                  # Generated by the pipeline
    ├── barcode77_read_stats.csv
    ├── nanoplot/barcode77/    # NanoPlot QC output
    └── plots/
        ├── gc_content_distribution.png
        ├── read_length_distribution.png
        ├── mean_quality_distribution.png
        └── summary_statistics.txt
```

## How to Run

### Prerequisites

- [Conda](https://docs.conda.io/en/latest/miniconda.html) or [Mamba](https://mamba.readthedocs.io/) installed.

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/Case-Study-Mini-Bioinformatics-Pipeline-Reporting.git
cd Case-Study-Mini-Bioinformatics-Pipeline-Reporting
```

### 2. Create and activate the Conda environment

```bash
conda env create -f environment.yml
conda activate snakemake
```

### 3. Run the pipeline

```bash
snakemake --cores all
```

This will:
1. Run **NanoPlot** — a QC tool designed for long-read data — producing read quality, length distributions, and summary statistics.
2. Run **read_stats.py** — a custom Python script that calculates GC content (%), read length, and mean Phred quality score for every individual read, saved as a CSV.
3. Run **visualize.py** — generates histogram plots for all three metrics with KDE overlays and annotated mean/median lines, plus a summary statistics text file.

### 4. View results

All outputs are in the `results/` directory:
- `results/nanoplot/barcode77/` — NanoPlot HTML report and plots
- `results/barcode77_read_stats.csv` — Per-read statistics table
- `results/plots/` — Distribution plots (PNG) and summary statistics (TXT)

| Step | Tool / Script | Purpose |
|------|---------------|---------|
